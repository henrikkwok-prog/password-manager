# 密码管理器

一个安全、简洁的跨平台密码管理应用，支持鸿蒙和安卓。

## 功能特性

- 🔐 **AES-256-GCM 加密** - 所有密码本地加密存储
- 🖐️ **生物识别解锁** - 支持指纹/面容识别
- 🔑 **密码生成器** - 自动生成强密码
- 💾 **备份与恢复** - 加密备份导出/导入
- 📱 **深色模式** - 优雅的暗色界面

## 快速开始

### 1. Fork 本仓库
点击右上角的 **Fork** 按钮，复制到你的 GitHub 账号。

### 2. 等待自动构建
- 进入你 Fork 的仓库
- 点击 **Actions** 标签
- 等待构建完成（约 5 分钟）
- 在 Artifacts 中下载 APK

### 3. 安装到手机
- 将 APK 传输到手机
- 允许安装未知来源应用
- 安装完成

## 手动构建

```bash
flutter pub get
flutter build apk --release
```

## 安全提示

⚠️ **务必备份主密码** - 主密码丢失将无法恢复数据

⚠️ **首次启动设置** - 请设置一个强主密码用于加密所有数据

## 技术栈

- Flutter 3.16+
- Dart
- SQLite (sqflite)
- AES-256 加密
